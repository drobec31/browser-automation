## Project Overview

A general-purpose Playwright CLI toolkit for browser automation: web app testing, screenshots, scraping, and task automation. Written in TypeScript, using the `@playwright/test` runner with real Google Chrome (`channel: 'chrome'`).

## Common Commands

```bash
# Install dependencies and browsers
npm install
npx playwright install chrome

# Run all scripts/tests
npx playwright test

# Run a single file
npx playwright test tests/example.spec.ts

# Record browser actions and generate code
npx playwright codegen https://example.com

# Take a quick screenshot from CLI
npx playwright screenshot --browser chrome https://example.com screenshots/out.png

# Open visual test runner (UI mode)
npx playwright ui

# Show last test report with traces
npx playwright show-report
```

## Architecture

All automation scripts and tests are `.spec.ts` files in `tests/`. There is no distinction between "scripts" and "tests" — both use `@playwright/test` APIs and are run the same way.

- `playwright.config.ts` — browser config (`channel: 'chrome'`), output dirs, screenshot-on-failure
- `tests/` — all automation files; add new `.spec.ts` files here
- `screenshots/` — manual screenshots (`page.screenshot({ path: 'screenshots/...' })`)
- `test-results/` — auto-captured failure screenshots and traces (generated by Playwright)

## Key Conventions

- Use `channel: 'chrome'` (real Chrome) rather than the default bundled Chromium
- Manual screenshots path to `screenshots/<name>.png`
- `test-results/` is gitignored — it's ephemeral output
